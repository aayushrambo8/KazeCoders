# URL-Based HTTP Attack Detector


## What it does
- Ingests PCAP files
- Extracts HTTP requests
- Normalizes and decodes URLs
- Detects and classifies URL-based attacks
- Identifies attack attempts vs successful intrusions
- Exports results to CSV and JSON


## Supported Attacks
- SQL Injection
- Cross-Site Scripting (XSS)
- Command Injection
- Directory Traversal
- SSRF
- LFI / RFI
- HTTP Parameter Pollution
- XXE
- URL Spoofing
